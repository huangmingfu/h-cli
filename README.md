## 全局安装
```sh
npm install -g hmf-cli
```

## 交互式选择模板
```sh
hmf-cli create  
# or  
hmf-cli create my-project
```

## 列出所有可用模板
```sh
hmf-cli list
```

## 直接指定模板类型（type）
```sh
hmf-cli create my-project -t react  
# or  
hmf-cli create my-project --template react
```

## 查看当前版本
```sh
hmf-cli -v
# or
hmf-cli --version
```

## 卸载
```sh
npm uninstall -g hmf-cli
```
